# Codex Workshop

Hands-on AI@GSB workshop site for teaching OpenAI Codex as an AI deployment
partner. The workshop walks participants through setup, a Codex cheat sheet,
a first local build, a full OpenAI API interview coach app, and bonus browser,
docs, spreadsheet, skills, and plugin workflows.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Scripts

```bash
npm run lint
npm run build
```
